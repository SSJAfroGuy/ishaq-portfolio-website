Information about the project

What happened

Last Cloudia held a Create-O-Rama event in 2022 where the players could create their own content related to the game, these submissions were suggested to be pictures or videos, so of course I decided to make a video game for this project. I worked with a friend on this gmae who provided the amazing art that was used

Plans

I honeslty forget how long we had for this project, however not too much time was put into it during the event, my idea for the project was mostly to create a base which could be worked on in the future as I have a very ambitions goal to turn this into a fully fledged game utilising all chracters in the official Last Cloudia game which would include over 100 unique characters, both playable and as bosses, whilst also making this a fully functional RPG which adds depth to the gameplay. For now it's just a small showcase of how the boss fights will work - There are plans to add waves of mob like enemies for the player to defeat for experience to add more stats and skills in the future

Controls

Arrow Keys = Move

Spacebar = Shoot

Alt + F4 = Close game (You need to do this if you win)

Things to note

The boundaries don't work vertically, so try not to go too far up or down

There is no main menu or pause button, you are thrown right into the game
